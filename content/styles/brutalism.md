# Brutalism (Web)

**Definition**: An anti-design aesthetic emphasizing raw, unstyled HTML elements, default system fonts (like Times New Roman or pure monospace), unaligned components, harsh native browser blue for links, and an intentional disregard for conventional UX/UI best practices. It aims to look "ugly," unfinished, or purely functional.

**History**: Borrows its name from the architectural movement of the 1950s—which celebrated raw, exposed concrete (*béton brut*). On the web, it emerged around 2014–2016 as a reaction against the overly polished, template-driven, "corporate" feel of modern web design (Flat 2.0 and Material). It was a return to the early 1990s web aesthetic, but done intentionally by highly skilled designers as an artistic statement.

**Variations**:
- **Purist Brutalism**: Pure, unstyled HTML. Looks exactly like a web page from 1993. No CSS positioning, raw anchor tags, default browser margins.
- **Graphic Brutalism**: Complex, chaotic layouts that *look* broken or amateurish (overlapping elements, stretched images) but actually require complex CSS and JS to achieve.
- **Utilitarian Brutalism / High-Tech Utility**: Extremely dense, grid-based, monochrome layouts focusing purely on data density. Heavily relies on tables, monospace fonts, and a "terminal" or command-line aesthetic.
- **Soft Brutalism**: Brutalist principles (exposed grids, systemic typography, raw imagery) but organized cleanly so it remains legible and usable for commercial purposes.

**Use Cases**:
- **Art & Architecture Portfolios**: Where the work demands an interface that doesn't feel like a standard "tech" product.
- **Underground Music & Fashion**: To signal authenticity, rawness, and an anti-commercial stance.
- **Developer Tools**: Utilitarian variants are used for dev-centric tools where speed, data density, and a "hacker" aesthetic are valued over marketing polish.
- **Editorial Deep Dives**: Some digital magazines use Brutalism for specific articles to break the reader out of their standard consumption habits.
